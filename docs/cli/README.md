SF-ADMIN-CLI是一个提供交互式命令行创建[sf-nest-admin](https://github.com/hackycy/sf-nest-admin)以及[sf-midway-admin](https://github.com/hackycy/sf-midway-admin)的工具。

为你更快速的进行上手开发，CLI 提供：

- 快速创建干净的 Nest 或 Midway版的项目模板
- 快速导入SQL脚本
