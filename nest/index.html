# 介绍

基于NestJs + TypeScript + TypeORM + Redis + MySql + Vue + Element-UI编写的一款简单高效的前后端分离的权限管理系统。具有：

  • 前后端请求参数校验
  • JWT 认证
  • 基于 NestJs 框架，内置了基础的中间件支持（用户认证、访问日志、请求追踪等）
  • 用户权限动态刷新
  • 代码简单，结构清晰

环境要求

  • Node.js 12.x+
  • Typescript 2.8+
  • MYSQL 5.7+
  • Redis 5.0+
